Parallelized Square Matrix Multiplication (Java)

Overview
This project implements square matrix multiplication using parallelized computation to leverage multicore architectures and improve CPU utilization. The implementation is written in Java and follows a Maven-based structure.

Features
✅ Parallelized Computation: Optimized for multi-threaded execution.
✅ Performance Optimization: Designed to maximize CPU usage on multi-core systems.
✅ JUnit Testing: Validated results against a well-known scientific computation software.
✅ Maven-Based: Easily build, test, and package with Maven.

Installation
Prerequisites
Java 17+
Maven
Git

Clone the Repository

git clone https://github.com/souheilredjeb/Lincompute.git
cd Lincompute

Build the Project

mvn clean install

Testing

mvn test
